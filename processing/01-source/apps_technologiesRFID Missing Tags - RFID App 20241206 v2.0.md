Missing RFID Tags

What departments need RFID tags attached?
The following departments are RFID enabled:

Apparel
- Women's, Fashion accessories, Men's, Kids, Active, Footwear & Footwear accessories.

General Merchandise
- Bed Linen (Sheeting & Pillowcases, Quilt Covers & Sets, 
- Blankets & Throws, Comforters, Cushions)

Note: Items with EPC labels do not need to be tagged

How do I print RFID stickers?
• Open RFID App, select Print, scan barcode or enter keycode, choose printer, input quantity and click the printer icon
• Tear sticker from printer, check the keycode matches and place RFID sticker on the Kimble tag

What if an item is missing the kimble tag?
• Open RFID App, select Print, enter keycode / style code / description
• Select correct size
• Select the printer, input quantity and select printer icon
• Whilst wearing gloves, attach the Kimble tag to product using the needle gun as per the below
