5S Standards

Sort
Get rid of unnecessary items

Shine
Clean up the workplace

Set in Order
A place for everything and everything in its place

Standardise
Set standards

Sustain
Maintain standards

The 5s process involves assessing everything present in a space, removing what is unnecessary, organising things logically, performing housekeeping tasks, and keeping this cycle going; organise,clean,repeat
