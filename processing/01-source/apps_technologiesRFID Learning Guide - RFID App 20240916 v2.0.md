RFID App
Learning Guide

This learning guide will focus on the application used to perform all functions relating to RFID, tagging, locating inventory and managing manual tasks. You will be taken through how to login and general navigation to familiarise yourself with the user interface.

Signing in

1. On the PDT select the RFID app 
- the login screen will appear

2. To sign in, enter your username and password
• To get your username and password, speak to a member of the management team
• If you can't login, please call Kmart support on:
   • Australia: 1800 033 534
   • New Zealand: 0800 944 555

2. Count capture
• As part of the daily schedule, you might need to complete a manual hand scan of areas that Tory can't access

3. Product lookup
• Search for products using either the barcode, keycode, description or style ID

4. Print
• To print an RFID tag for an item, you will be able to search using a:
   - Keycode, description or style ID
• Once you have found the item, connect to the RFID printer to print a new tag

5. Backfill
• To action digital backfill tasks, this section will show you the tasks per department and how many items require backfill

## Page 3

Navigating the RFID app continued

6. Relocate
• The relocate function is used to move unique tags (EPC's) between different storage areas
• Stores can use this function to relocate tags from 'offsite' back into a storage area in their store

7. Inventory summary
• As part of the manual handscan process, 'Inventory Summary' is the last step to book, finalise and send the data for digital tasks to be created

8. Offsite
• The offsite tile is available for stores to use if apparel stock is required to be sent to an offsite or held at the DC
• This will ensure the units captured will remain in inventory for SOH adjustments
