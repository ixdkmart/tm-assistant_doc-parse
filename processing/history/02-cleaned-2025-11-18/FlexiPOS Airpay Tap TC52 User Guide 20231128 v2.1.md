# FlexiPOS - AirPay Tap

The TC52 PDTs (these PDTs commence with a 200..PDT number) have been updated with Airpay Tap. This allows all FlexiPOS transactions to be processed by tapping the EFT card directly on the back of the PDT when selecting 'Pay @ Card Reader' to complete the transaction.

Note:
The AP3 pinpads or TC51s will no longer be used for FlexiPOS transactions.

When using FlexiPOS for the first time after the upgrade to Airpay Tap, an initial process will need to be performed to verify permissions required to perform Airpay tap transactions.

There is no transaction limit on FlexiPOS Airpay Tap transactions.

The 'Tap and Go limit' (depending on the customer's bank) is $100.00. For transactions over this amount, a new 'Enter Pin' screen will be displayed allowing the customer to enter their pin to complete the transaction.

An AP3 (pinpad) is not required when performing an AirPay tap transaction.

AirPay Tap can only be used on a TC52 PDT. These PDTs commence with 20, e.g., KMPD201323.

# AirPay Tap Verification

Each TC52 PDT must be verified after the AirPay Tap upgrade.

When logging into FlexiPOS, a message will be displayed to 'Verify Now'. Select 'Verify Now' and then the back arrow.

The 'Permissions Required' screen will be displayed. Select 'Done'.

The transaction can now be completed using AirPay Tap:
Select 'Pay @ Card Reader'.
Tap the card on the rear of the PDT towards the top.
Payment will be processed and a 'Transaction Approved' message will be shown if approved.

# FlexiPOS Detailed Guide

From the PDT, select 'FlexiPOS'.
Enter the employee ID.
Enter the same password used for other PDT apps.

First-time login requires assigning a portable printer for printing receipts.

A prompt for 'Select a printer' will appear. Select the appropriate portable printer.

A confirmation message 'Default Printer set successfully. Check your correct printer roll loaded' will appear.

The FlexiPOS welcome screen will be shown.

(Various screenshots show payment screens, printer selection, welcome screens, and transaction approval.)

# Printer Register Roll

A full-size register roll does not fit inside the printer. Use the register rolls sized specifically for FlexiPOS, which have been sent to stores.

# Logging into FlexiPOS on Multiple PDTs

If a team member logs into FlexiPOS on one PDT and then attempts to log in on a second PDT, a message will appear: "You are currently logged in to device e.g., KMPD04803. Would you like to logout from the device, yes or no?" Select 'Yes' to log in.

If the same team member logs into another PDT after being logged out, a message will state: "You have been logged out of this device because you have logged into KMPD......".

# Performing a FlexiPOS Transaction

Start a transaction by selecting 'Start Transaction'.

The 'Cart' screen will be shown. Scan items, add items, capture team member discounts, flybuys, or manually enter items.

To remove an item, select the trash can icon next to the item, then choose 'Yes Remove' to confirm.

# Performing a Price Change on an Item

Select the 'pencil' icon next to the item.
Enter a new price.
Select the 'Price change reason'.
Choose 'Apply'. The original and new override prices will be updated on the cart.

# Performing a Price Change on Multiple Quantities

Select the 'pencil' icon next to the item.
Enter a new price.
Change the quantity to the required amount.
Select the 'Price change reason'.
Choose 'Apply'.

# Item Lookup

The item lookup feature assists if items have lost tags or barcodes.
Search can be performed by keycode, style, or description.
Description search returns items from newest to oldest.
A single item can be selected and added to the transaction.

Search by keycode:
On the 'Search' screen, select 'Search by keycode'.
Enter the keycode (e.g., 7017414) and select 'Search'.
Select '+' to add the item to the cart.
A message will confirm the item was added successfully.

Search by style code:
Select 'Search by style code'.
Enter the style code (e.g., 33803) and select 'Search'.

Search by description:
Select 'Search by description'.
Enter a description (e.g., Monopoly).
Select 'Search'.
Select '+' to add items from the search results to the transaction.
Use '+' or '-' to modify the quantity of an item.
Alternatively, select 'Quantity' to modify and submit a new value.

# Cancelling a Transaction

From the 'Cart' screen, select the arrow next to the total amount.
Choose 'Cancel transaction' and confirm with 'Yes'.

From the 'Payment' screen, select the back arrow.
Choose 'Return to Cart' to add more items or 'Cancel Transaction' to cancel.
Confirm 'Cancel Transaction', and the cart will be cleared with a message indicating the cart is empty.

# Completing a Transaction - EFT Only

After adding items, view the basket and select 'Pay'.
The amount due and any team discounts will be shown.
Select 'Pay @ Card Reader'.
Tap the card on the rear of the PDT toward the top.
Payment will process, and 'OK' will indicate approval.
A receipt will be printed. Press 'New Transaction' or scan an item for a new sale.
A duplicate receipt can be printed at the end by selecting 'Reprint'.

# Transactions Over Limit Requiring a PIN

If a transaction exceeds the Tap & Go limit, tap the card on the rear of the PDT.
A 'Enter PIN' screen will be shown.
The customer must enter their PIN and confirm with the green tick.
The transaction will then complete with a receipt.

# Digital Receipt

A digital receipt can be sent to the customer's mobile.
Select 'Share' and enter the mobile number.
The customer will receive a message with a link to generate an OTP.
Enter the OTP to view the digital receipt.

# WIFI Coverage Issue

If a receipt is not sent, it may be due to WIFI coverage issues.

# Total Sales Values

When logging out and back in with the same credentials, total sales, transactions, and averages are displayed on the welcome screen. These values reset at midnight.

# Age Restricted Items

Items with age restrictions will prompt a message for approval before adding to the transaction.

# Withdrawn Items

Scanning a withdrawn item will display a message; confirm to prevent adding it.

# Sale of Bikes

Scanning a boxed bike prompts, 'Would you like to add bike assembly fee?' Selecting 'Yes' adds the fee. Scanning a bike assembly barcode adds the fee automatically.

# Settings Menu

Accessed via the top right icon, options include setting printers, reprinting receipts, pairing/unpairing card readers, resetting Bluetooth, updating card readers, and tools.

# Set Printer

Select 'Set Printer', then 'WiFi' and 'Search'.
Choose a printer and confirm.
Return to the previous screen by selecting the close (X) button.

# Reprint Last Receipt

Select 'Reprint' to print the last receipt. A 'Receipt printed successfully' message confirms.

# Void Transaction with QR

Allows voiding a transaction and generating a QR code receipt to complete at a manned register.
Select 'Void Transaction with QR', and a message indicates success.
The QR code can be scanned at any manned register, displaying transaction details for completion.

# Logout of FlexiPOS

From the welcome screen, select the 'Power' option.
Confirm by selecting 'Yes' on the logout prompt.

# Reporting - Pos To WCR

A POS To WCR report runs automatically every Monday at 04:56am. Stores must perform their store close beforehand.
This report includes combined figures from registers, SCOs, and FlexiPOS.
Stores import these figures into their weekly cash report as usual.
A copy is saved to G:\Kmart Line Management.

# Store-Specific Report Sample

Displays financial and transactional details for store 1902, including GST outputs, sales, tender details, department sales, and inventory data.

# Reporting - SATR Recap and Department Totals

Daily at 1am, reports on FlexiPOS figures are printed.
Includes store accounting totals, department totals, and saves copies in designated folders.

# Store and Department Reporting

Provides detailed sales, returns, voids, and discount figures per department.
Total sales and other metrics are summarized.

# Store Accounting Totals

Summarizes gross sales, net sales, items sold, EFT purchases, voids, discounts, taxes, tender types, and total figures.
All figures are from the previous period specified.

# Price Override Report

Items with price overrides during transactions are listed, showing item name, POS code, old and new prices, and comments.

# Electronic Journal - FlexiPOS

Accessible via the Escape menu on the Back Office.
Requires login with username and password.
Search transactions using multiple parameters, including employee ID, keycode, card number, discounts, date, or transaction number.

# Viewing and Exporting Transactions

Transactions can be viewed on-screen or exported to Excel for further analysis.
Transaction details include ID, date/time, store, register, status, amount, and product info.
A detailed view shows full transaction information and product details.

# Cumulative Hourly Sales

The hourly sales report does not include FlexiPOS sales.
