# Pallet Barcode Guide – Template Set Up

## Template Set Up
Print the required quantity of the template.  
Print the required pallet barcodes and attach 2 identical barcodes to each template, referring to the pallet barcode printing guide.  
Laminate each template so they can be reused.  
Store the template in a folder near the decant area with one bulldog clip per template.

# Pallet Barcode Guide – During Decant & Sequence

## During Decant
When creating a pallet in decant, attach the pallet barcode template using a bulldog clip.  
Use a whiteboard marker to write the decant date and pallet number.  

## During Sequence
The team member fills the pallet.  
Scan the barcode to complete.  
The runner or sequence manager confirms the barcode has been scanned.  
Once scan is confirmed, the runner or sequence manager removes the template.  
Store the template in the folder for the next decant.
