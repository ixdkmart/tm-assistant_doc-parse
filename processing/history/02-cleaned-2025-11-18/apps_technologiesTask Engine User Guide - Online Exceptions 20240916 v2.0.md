# Task Engine Learning Guide

## Accessing the Task Engine application
This application can only be accessed on a TC52 or an upgraded TC51 PDT device. These can be identified by their PDT numbers that will start with either KMPD20 or KMPD30.

## Logging in
Select the Task Engine 'Tasks' application from the PDT. Enter your employee number followed by @kmart.com.au. Enter your password.

## Creating a PIN code
After your first login each day, create a 4-digit PIN code that will last for 24 hours. Create and enter a 4-digit PIN. Press 'Confirm'. The Task Engine main screen will show your PIN has been accepted.

## Re-entering PIN after idle time
If the PDT is idle for 25 minutes, your PIN will need to be used to access the Task Engine. After idle time, enter your 4-digit PIN to access the Task Engine.

## Forgot PIN (PIN reset)
To reset your PIN, select 'I forgot my PIN' and log back in with your employee number followed by @kmart.com.au.

## Online Exception - completing a task
To filter tasks by World, select the filter function. Choose the desired World and press 'Apply'. Tasks shown will be from that World only. Press 'Clear' to reset.

Open a task to complete by selecting its card. Online exceptions should be completed according to the current process, ensuring 'Confirm' is pressed when finalising each adjustment. When the task is complete, you are automatically returned to the task engine home screen.

## Logging out
Select your initials from the top right, then choose 'Log out'. The screen will show that all tasks are complete.

## Troubleshooting
Follow these steps for the following error messages:

Error - something went wrong on home screen
Pull down the app to refresh. If the error persists, log out and log back in. If still unresolved, reboot the PDT. If rebooting does not clear the error, contact the IT Service Desk.

Error - lost connection on home screen
Pull down the app to refresh. Ensure you are in network range with Wi-Fi bars on the PDT. If the error continues, log out and log back in.

Error - lost connection with no tasks displayed on home screen
Pull down the app to refresh. Ensure network range and Wi-Fi bars. If unresolved, log out and log back in.

Error - 'something went wrong' with no tasks on home screen
Log out and log back in. If the error persists, reboot the PDT. If rebooting does not resolve it, contact the IT Service Desk.

Forgotten password
Visit https://portal.office.com on a desktop computer to reset your password. If your password is correct but login issues persist, contact the IT Service Desk. If the issue cannot be resolved, contact the IT Service Desk at:
Australia: 1800 033 534
New Zealand: 0800 945 995

To reset your Microsoft AD password, visit https://portal.office.com on a PC.
