# RFID App Learning Guide

This learning guide explains how to use the RFID app for all functions related to RFID, tagging, locating inventory, and managing manual tasks. It covers login and navigation to help you become familiar with the user interface.

# Signing in

1. Select the RFID app from the PDT.
2. The login screen will appear.
3. To sign in, enter your username and password.
4. Obtain your username and password by speaking to a member of the management team.
5. If you cannot log in, call Kmart support:
   Australia: 1800 033 534
   New Zealand: 0800 944 555

# Navigating the RFID app

1. Once signed in, the main screen will appear.

# Navigating the RFID app continued

## Count capture

1. Complete a manual hand scan of areas Tory cannot access as part of the daily schedule.

## Product lookup

1. Search for products using the barcode, keycode, description, or style ID.

## Print

1. Search for an item using the keycode, description, or style ID.
2. Connect to the RFID printer to print a new tag.

## Backfill

1. View backfill tasks per department and the number of items requiring backfill.
