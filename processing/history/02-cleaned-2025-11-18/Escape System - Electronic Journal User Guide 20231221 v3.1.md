Escape System Electrical Journal User Guide

Introduction

The Escape system is an electronic journal used to search for transactions and data entered through the registers such as cash transactions, void transactions, refunds, etc.

This guide explains how to use the Escape system models to search for specific data.

When you first access the Escape system, you are taken to the main screen and select the search option to begin.

[Menu options shown:]
- Review
- Search
- Reports
- DAVE
- Exit

## Page 4
Escape System Electrical Journal User Guide

Model

The search function allows you to filter for various parameters across all operators and registers. This includes sales for certain dollar values, no sales, missing or excessive codes, APNs including department #95, cash-only sales, credit card sales, discounts, promotions, voids, suspended sales, laybys, cancellations, frequent card numbers, non-merchandise sales, price overrides, refund exchange vouchers, positive sales on refund register, and unauthorised operator codes.

Any register activity can be tracked using the system. Gaining experience will help save time in investigations and identify fraud.

Load Model

Models help in initial setup by pre-selecting parameters like:

- Transactions between $100 and $150  
- Cash tender or charge between $50 and $60  
- Incomplete voids  
- Laybys: sales between $20 and $30, customer cancels, store cancels, voids  
- Refunds: price corrections, positive sales  
- All transactions by operator #652  
- EFTPOS with cash out  
- Suspended sales  
- Department #95 transactions  
- No sale transactions by operator #652  
- Final layby with zero GST  
- Initial laybys with 8 or more items  
- Suspended transactions with zero total

To load a model, open the search, and click 'Load Model'. Choose the desired model. Search details like register, operator, transaction number can be modified after loading.

Printing Transactions and Reports

To print a transaction or report:

From the search results, select 'Print', then 'Print Font'. Choose the size and confirm.

In the same results box, select 'File', then 'Print', and choose 'Print Transaction' or 'Print Reports'.

When printing reports, review the page count to avoid overloading. Cancel or customize the output as needed.
