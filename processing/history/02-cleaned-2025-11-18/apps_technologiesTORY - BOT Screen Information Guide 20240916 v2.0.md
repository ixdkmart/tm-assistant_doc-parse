## Bot LED

## Screen Information Guide

### Bot name

Consists of TORY and the last 3 digits of its serial number.  
Format: TORY-XXX, where X = 0-9.

### Battery power percentage

Unplugged: +1.5 to 3.0A.  
Charging: -5 to -10A.  
Fully charged: -0.4 to 0.5A.

### Battery percentage

100% corresponds to 0%.

### Inventory status

The current inventory status remains the same after a run until the next run.  
Idle indicates the application has started.  
Running indicates an ongoing inventory run.  
Finished indicates the inventory was successfully completed.  
Interrupted indicates the inventory run had to be interrupted.

### Run duration

Displayed as [HH:MM:SS].

### Hardware status

OK.  
BLOCKED indicates the motor brake is active; do not push Tory manually.  
OK | FREE RUN indicates the motor brakes are inactive; manual movement is allowed.  
BUMPER indicates inability to drive, possibly due to bumper being pressed.  
EM STOP indicates the emergency stop is engaged.  
INTERNAL ERROR indicates an internal hardware error.

### Time

Only updates when the application is running.  
Displayed as [HH:MM:SS].
