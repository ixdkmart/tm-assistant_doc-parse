FLEXIPOS - AirPay Tap

The TC52 PDTs (these PDTs commence with a 200..PDT number) have been updated with AirPay Tap.
This enables all FlexiPOS transactions to be processed by tapping the EFT card directly on the back of the PDT when selecting 'Pay @ Card Reader' to complete the transaction.

Note:
The AP3 pinpads or TC51s will no longer be used for FlexiPOS transactions.

When using FlexiPOS for the first time after the upgrade to AirPay Tap, an initial process will need to be performed to verify permissions required to perform the AirPay tap transactions.

There is no transaction limit on FlexiPOS AirPay Tap transactions.

The 'Tap and Go limit' (depending on the customer's bank) is $100.00.
For transactions over this amount, a new 'Enter Pin' screen will be displayed allowing the customer to enter their pin to complete the transaction.

There is no AP3 (pinpad) required when performing an AirPay tap transaction.

AirPay Tap can only be used on a TC52 PDT.
These PDTs commence with 20, e.g., KMPD201323.

Verify the AirPay Tap (actioned once on each TC52 PDT):

Each TC52 PDT will need verification after the AirPay Tap upgrade.

When logging into FlexiPOS, a message will be displayed to 'Verify Now.'

Select 'Verify Now,' then select the back arrow.

The 'Permissions Required' screen will be displayed.

Select 'Done.'

The transaction can now be completed using AirPay Tap.

Select 'Pay @ Card Reader,' then tap the card on the rear of the PDT towards the top.

Payment will be processed and a 'Transaction Approved' message will display if approved.

# Page 4

Printer Register Roll

A full-size register roll will not fit inside the printer.

Use specific stationer rolls sent to stores for FlexiPOS transactions.

Logging into FlexiPOS on multiple PDTs:

If a team member logs into FlexiPOS on one PDT and then tries on a second PDT, a message will appear:

'You are currently logged in to device e.g., KMPD04803. Would you like to logout from the device, yes or no?'

Select 'Yes' to log in to FlexiPOS.

If the same team member logs in again on another PDT, the message will be:

'You have been logged out of this device because you have logged into KMPD......'

# Page 5

Performing a FlexiPOS Transaction:

Select 'Start Transaction' to begin.

The 'Cart' screen will be displayed.

From the Cart, scan items to add to the transaction.

Capture team member discounts and Flybuys.

Manually enter items by selecting 'Key in Items.'

To remove an item, select the trash can option next to the item.

A confirmation message will appear. Select 'Yes Remove.'

The item will be removed from the cart.

# Page 6

Performing a Price Change on an Item:

Select the 'pencil' icon next to the item.

A dropdown list for the 'Price change reason' will appear.

Enter a new price, select the reason, then select 'Apply.'

The 'Original Price' and the overridden price will be updated on the cart.

[Screenshots show items like GAME MONOPOLY ($29.00), UNO CARD TEST ($22.50), and SMARTIES CARTON 50G ($1.75), demonstrating price change and removal.]

# Page 7

Price Change for Multiple Quantities:

Select the 'pencil' icon next to the item.

A dropdown will show with total quantity.

Enter a new price, change quantity if needed.

Select the reason, then 'Apply.'

The change applies to the specified quantity only.

[Screenshots depict a cart with items, including UNO CARD TEST priced at $7.50 and SMARTIES CARTON 50G with overridden price, total $23.50 for 4 items.]

# Page 8

Item Lookup:

FlexiPOS has an item lookup feature to assist with lost tags or barcodes.

Search by Keycode, Style, or Description.

Description searches from newest to oldest.

Select an item and add to the transaction.

Searching by Keycode:

From the search screen, select 'Search by keycode.'

Enter the keycode (e.g., 7017414), then select 'Search.'

Add the item to the cart by selecting '+'.

Message: 'Item added to the cart successfully!'

Searching by Style Code:

Select 'Search by style code.'

Enter the style code (e.g., 33803), then search.

# Page 9

Select 'Search.'

Message: 'Item added to the cart successfully!'

# Page 11

Cancelling a Transaction:

From the 'Cart' screen:

Tap the arrow next to total amount, then select 'Cancel transaction.'

Confirm by selecting 'Yes.'

From 'Payment' screen:

Tap back arrow to return to cart or cancel.

Select 'Cancel Transaction' to cancel the transaction.

A message 'Your cart is empty' will appear, and a new transaction will start.

# Page 12

Completing a Transaction (EFT Only):

After items are in the cart, select 'Pay' to finalize.

View items to confirm.

Select 'Pay @ Card Reader,' then tap the card on the rear of the PDT.

Payment will process, with 'OK' if approved.

A receipt will be printed on the connected printer.

Select 'New Transaction' or scan items to start again.

A duplicate receipt can be reprinted.

# Page 13

Transactions over the Tap and Go Limit requiring a PIN:

If over $100, the 'Enter PIN' screen will appear.

The customer must enter their PIN and confirm.

Payment continues normally, with a receipt produced.

# Page 14

Digital Receipt:

Select 'Share,' then enter the customer's mobile number.

A text message with a website link will be sent.

The customer can generate an OTP via text message to view their digital receipt.

WIFI coverage issues may prevent sending receipts.

# Page 15

Total Sales Values:

When a team member logs out and back in, sales totals are displayed on the 'Welcome' screen.

Includes total sales value, total transactions, average transaction time, and average transaction value.

Values increment with each login and reset at midnight.

# Page 16

Age Restricted Items:

A message will appear for items with age restrictions.

Team members must approve before adding these items.

Withdrawn Items:

Scanning a withdrawn item will display a message that the item is withdrawn from sale.

Confirm the message; the item will not be added.

Scanning a boxed bike prompts: 'Would you like to add bike assembly fee?'

Settings Menu:

Accessed via the icon on the top right.

Allows setting printers, reprinting receipts, pairing/unpairing card readers, device resets, updates, and tools.
