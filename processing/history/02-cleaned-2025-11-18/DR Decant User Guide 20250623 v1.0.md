# Page 2

Accessing decant cage scan

1. Select the menu option.
2. Select Cage scan.
3. You can either create a new decant group or see decant groups listed.

The screen shows the options: Decant, Cage scan, Ad-hoc replenishment, Loadfill.

v3.5

# Page 3

Creating a decant group

1. Select create decant.
2. The date and time option will be shown.
3. Select the date; today’s date will be auto-selected.
4. Select the time; today’s time will be auto-selected. Select Create Decant.
5. The decant group will be displayed. Select the decant group to start scanning equipment.

The interface shows steps for creating a decant group, including date and time selection, with a final screen showing "Decant 19/07 7am" with total cages, rails, and pallets initially at zero.

# Page 4

Decant cage scan - scanning equipment

Step-by-step instructions:

1. Scan the barcode of the completed equipment using the PDT scan button.
   The initial screen shows: "Decant 19/07 12pm" with total 0 cages, 0 rails, 0 pallets, and an example cage UN20053373.
2. Select the area by choosing from options: Home, Kids, Clothing, Event.
3. Equipment will be added to the decant group. Continue to repeat the steps as you add more equipment.
4. The confirmation message "Added to H1" will be shown, with total cages increasing and the complete count button available.

# Page 5

Decant cage scan – editing equipment

Step-by-step instructions:

1. Scan the barcode of the equipment you wish to edit using the PDT scan button.
2. A popup message will display the equipment details; select edit.
3. Select the world.
4. Select the area.
5. Equipment will be updated in the decant group.

# Page 6

Decant cage scan – deleting equipment

Step-by-step process:

1. Scan the barcode of the equipment you wish to remove using the PDT scan button.
2. A popup message will display the equipment details; select Remove.
3. Confirm removal by selecting Remove again.
4. A popup message will show that the equipment has been removed.

The screen shows:
- Decant 19/07 7am
- Total 80 cages, 15 rails, 3 pallets

Categories:
- Home: 49 cages
- Kids: 20 cages, 5 rails

- Event: 1 cage

"Complete count"

# Page 7

Completing a decant group

1. Select Complete count.
2. A popup will ask: "Is the load fully decanted?"
3. If Yes, select Yes and then select Complete count.
4. If No, select No and then select Complete count.
5. The decant group will show as Completed.

Interface shows:
- Decant 19/07 7am
- Total 80 cages, 15 rails, 3 pallets
- Categories:
  - Home: 49 cages
  - Kids: 20 cages, 5 rails
  - Clothing: 10 cages, 10 rails, 3 plts
  - Event: 1 cage
