# Loadfill (Adhoc Replenishment) Policy and Procedure

## Loadfill Accessing Loadfill

Select the Menu option.

Select Loadfill under Adhoc Replenishment.

## Completing a cage, rail, or pallet

Scan the barcode on the equipment to be filled.

The screen will show "Scan barcode to start" with a "Let's go!" message.

Select Start loadfill cage.

The scanned location and H-Code will be displayed.

An "Edit area" option will be available.

Tap the "Start loadfill cage" button.

The equipment type and status will be shown as in progress.

The display will indicate the cage as "In progress," with the H-Code, current time, and progress.

Fill the scanned equipment type.

Once completed, the equipment type and area will show as done, with a message displaying "Nailed it. [Cage number] cage is done."

The process can be repeated for each equipment type to be filled.

## Completing multiple cages, rails, or pallets

Scan the barcode on the equipment to be filled.

The loadfill screen will indicate the progress and time.

Select Start loadfill cage for the next cage.

The scanned equipment type and H-Code will be shown, with status as in progress.

Fill the scanned equipment type.

Once completed, the equipment types will be listed as done.

A message such as "Great! [Cage number] cage is done." will be displayed.

This process continues until all equipment types are filled.

## Area not known

Scan the barcode on the equipment to be filled.

No area will be shown initially. Select Start loadfill cage.

Select the desired world area, such as Home, Kids, Clothing, or Event.

Select the specific area; options include C1: Womenswear, WUSSH; C2: Menswear, Basics; C3: Footwear; C4: Beauty & Accessories.

The equipment type will be associated with the selected area.

Select "Start loadfill cage" to proceed.

Scan the equipment type; the status will show as in progress.

Fill the equipment type.

Once completed, the system will show the equipment type and area as done, with a completion message.

## Previous equipment not scanned and completed

Scan a new equipment type while previous equipment is in progress.

A pop-up message will ask if the previous equipment type is done.

Select Yes or No.

If Yes, proceed to select next equipment.

The system will ask to start the loadfill cage for the new equipment.

Scanned equipment and H-Code will be shown, with status as in progress.

Fill the equipment type as instructed.

Once completed, the system will list all completed equipment types.

## Previous equipment not scanned and not done

Scan a new equipment type while previous equipment is in progress.

A message will ask if the previous equipment type is done.

Select No to remove it.

The previous equipment will be removed, and the new equipment will be in progress.

Proceed to fill the new equipment type.

Once completed, the system will confirm removal and show the new equipment as done.

## Screen messaging

If the equipment type has already been completed, the screen will show "Loadfill already completed" with a "Close" button.

If someone else is working on the cage, the screen will display a warning, "Someone is already working on this cage."

If the cage has been removed due to completion, a message will indicate "C2 cage removed as it is already completed."

## Loadfill screen information

The loadfill screen displays total equipment types with progress (e.g., Progress: 1/2).

Current equipment time is shown, counting down (e.g., 1 min / 0 min).

The area associated with equipment is indicated.

Equipment and barcode numbers are displayed.

Completed equipment times are shown (e.g., Completed in 14 mins).

Status indicators show "In progress" or "Done."

An input field allows entering the equipment barcode.

The timer displays 0 min when no cage is in progress.

All completed equipment types and their statuses are shown accordingly.
