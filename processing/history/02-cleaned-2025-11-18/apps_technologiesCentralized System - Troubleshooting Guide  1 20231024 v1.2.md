# Centralized System - Troubleshooting Guide

## Call Notification Missing - TC51 PDT

If the TC51 unit is ringing for an incoming call but there is no notification appearing to answer, do the following steps:

Open the Phone app on the TC51.  
On the top right of the app, select the three lines to access the menu.  
In the navigation pane, you will see the phone number or user who is calling you alongside the green answer button.  
Select the green answer button.  
You will be taken to the page where you can answer the phone.

## "Waiting for DHCP" Error

If a phone has reached any number above 200 on "waiting for DHCP", reset the phone settings as follows:

Select * on the phone.  
Enter the code 27238 and select #.  
Scroll down until you see the option "Reset Values".  
Leave the phone to boot until you are presented with the extension screen.
