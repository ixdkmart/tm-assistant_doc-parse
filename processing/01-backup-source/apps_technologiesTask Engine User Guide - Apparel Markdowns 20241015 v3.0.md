## Page 4

Apparel Markdowns - completing a task

1. On the Task Engine home screen select a markdown task

Team Members: Always select the top task card or tasks allocated from your manager, if you have task 'In progress' please select this task and complete prior you next task.

Manager Only: You will need to use the filter function to only show Markdown tasks.

2. From within the list view, select the first style. This will ensure that you follow the most efficient route within the task. Scroll down to see all styles listed if required.

Note: You cannot scan an item from this screen.

## Page 6

Apparel Markdowns - completing a task

5. Once the correct style/keycode has been scanned, the green tick will show and pop up 'keycode scanned'
6. Locate all the units in task
7. Apply the new prices to the kimble tag including shelf talker and press 'Done' to complete this style/colour task.
8. The next style to complete will appear, repeat the process until all styles are completed

Note: If you cannot find all the units in the task, you can still proceed to the next style colour, if you have a large variance please speak to your manager

Note: Please refer to the 'Smart Apparel Clearance Guide for pricing standards

## Page 7

Apparel Markdowns - completing a task

8. Press the 'Finalise Markdown' to complete this task

9. Once all styles have been marked you will be taken back to the list view into the 'Done' tab

Note: Please ensure that you check the 'To do' tab to ensure all styles/colours have been completed

9. A pop up will appear to confirm you want to finalise the task. Press 'Finalise' to confirm that this task is complete

Note: If you have styles/keycodes you were unable to find and they are in the 'To do' list, you will still be able to finalise the task.

10. You will be redirected back to the Task Engine home screen where a new markdown task can be started. Repeat the steps until all required task are completed

Note: If you have task 'In progress' please select this task and complete prior you next task.

## Page 8

Troubleshooting - Apparel Markdowns
Refer to these steps when troubleshooting the following error messages

Pop Up:
'Select style to scan' list view screen
1. You need to select the style pressing on the screen, which will take you to the style detail screen

Pop Up:
'Incorrect keycode scanned' list view screen
1. The incorrect keycode has been scanned
2. Check the item to ensure it is the correct keycode
3. Check the keycode list and ensure the keycode is listed in the task

Pop Up:
'Style already scanned' details screen
1. The style has already been scanned
2. Ensure that the items have been marked down and labelled
3. Press 'Done' to move onto the next style

Pop Up:
'Keycode already scanned' details screen
1. The keycode has already been scanned
2. Ensure that the items have been marked down and labelled
3. Press 'Done' to move onto the next style

## Page 9

Troubleshooting - Apparel Markdowns
Refer to these steps when troubleshooting the following error messages

Error message: 'Lost connection'
1. Ensure you are in network range and have Wi-Fi bars on the PDT
2. If error message is still displayed, log out and log back into the Task Engine

Once you are in network range a pop up 'you are back online' will be displayed

Error message: 'Lost connection'
1. Ensure you are in network range and have Wi-Fi bars on the PDT
2. If error message is still displayed, log out and log back in to the Task Engine

Note: Map will be grayed out and not available whilst in offline mode & More Info will not be available

Once you are in network range a pop up 'you are back online' will be displayed

Note: the 'More info' tab may show as loading whilst getting back online

If the application is offline, you will not be able to start or finalise a task; you will be able to continue a task if you have started

No Apparel Markdown tasks displayed in Task Engine
1. Please check the STT that there is scheduled markdowns for this week
2. Please ensure that TORY has had successful run above 90% run quality
3. Tasks created will be visible from Monday 3pm. (Melbourne Time)
4. If no tasks are shown for a successful run, please allow up to 1 hour and try again
5. If no tasks are displayed contact the IT Service Desk
