## Page 4

Loadfill (Allocated) - Editing a cage, rail, pallet

1. After you have filled the selected equipment type and area, you will need to scan the equipment barcode with the PDT

2. Select edit area

3. Select the world
- Options shown include: Home, Kids, Clothing, Event

4. Select the area
- Shows areas like C1, C2, C3, C4
- Includes categories like Womenswear, Menswear, Basics, Footwear, Beauty & Accessories

5. Pop up will show area has been updated. Select Complete loadfill cage

6. Completed equipment type and area will show as complete, pop up will also display as done.

Step-by-step process:

- Options include: C1 (Womenswear, W1/SH), C2 (Menswear Basics), C3 (Footwear), C4 (Beauty & Accessories)
