## Page 3

Creating a decant group

1. Select create decant

2. Date and time option will be shown

3. Select the date, todays date will be auto selected

4. Select the time, todays time will be auto selected. Select Create Decant

5. Decant group will be displayed. Select the decant group to start scanning equipment

Options:
- Home
- Kids
- Clothing

## Page 7

Completing a decant group

1. Select Complete count

2. Pop up message will display Is the load fully decanted?

3. Select Yes if all pallets have been completed and then select Complete count

4. Select No if all pallets are not completed and then select Complete count

5. Decant group will show as Completed
