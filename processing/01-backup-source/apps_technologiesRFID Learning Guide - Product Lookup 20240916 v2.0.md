Product lookup function

The RFID app will help you to check inventory quickly and easily whether there is a tag or not. If you need to find an item, you can use the "Show on Map" feature to locate the product in the back of house or on the shop floor.

Using the RFID app to check inventory

1. Log into the RFID app

2. Select the 'Product Lookup' option from the main screen

3. To find the right item either
• Scan the barcode or
• Enter using:
   • Keycode
   • Description 
   • Style ID
   • If you enter the relevant detail to search, click on the magnifying glass to bring up the item

4. If you have used a keycode, style ID or description, make sure you select the correct item
• Once you've scanned the barcode, the item will appear

5. Select the 'Show on map' link in the bottom left-hand corner to bring up the location map

6. The items will show on the map as green tags

7. Zoom in on the map using the "+" and "-" buttons or using your fingers, similar to how you'd zoom in on your phone
