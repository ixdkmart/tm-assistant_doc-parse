Accessing the Task Engine application
This application can only be accessed on a TC52 or an upgraded TC51 PDT device. These can be identified by their PDT numbers that will start with either KMPD20 or KMPD30.

Logging in
Please follow the steps below to log into the Task Manager

1. Select the Task Engine 'Tasks' application from the PDT

2. Enter your employee number followed by @kmart.com.au

3. Enter your password

Creating a PIN code
After your first log in each day a 4-digit PIN code can be created that will last for 24 hours

1. Create and enter a 4 digit PIN
2. Press 'Confirm'
3. Task Engine main screen showing your PIN has been accepted

Re-entering PIN after idle time
If the PDT is idle for 25 minutes your PIN will need to be used to access the Task Engine.
1. After idle time, enter your 4 digit PIN to access the Task Engine

Forgot PIN (PIN reset)
To reset your pin:
1. Select 'I forgot my PIN' and
2. Log back in with your employee number followed by @kmart.com.au

Troubleshooting

Refer to these steps when troubleshooting the following error messages:

Error - something went wrong on home screen
1. Pull down the app to refresh
2. If error message is still displayed, log out and log back into the Task Engine
3. If error message is still displayed after logging back in, reboot PDT
4. If rebooting the PDT does not clear the error message, contact the IT Service Desk

Error - lost connection on home screen
1. Pull down the app to refresh
2. Ensure you are in network range and have Wi-Fi bars on the PDT
3. If error message is still displayed, log out and log back into the Task Engine

Error - lost connection with no tasks displayed on home screen
1. Pull down the app to refresh
2. Ensure you are in network range and have Wi-Fi bars on the PDT
3. If error message is still displayed, log out and log back in to the Task Engine

Error - 'something went wrong' with no tasks displayed on the home screen
1. Log out of the Task Engine and log back in
2. If error message is still displayed after logging back in, reboot the PDT
3. If rebooting the PDT does not clear the error message, contact the IT Service Desk

Forgotten password
1. Visit https://portal.office.com on a desktop computer to reset your password
2. If your password is correct, but you still cannot log into the Task Engine application, contact the IT Service Desk

If the issue is unable to be resolved after following the troubleshooting steps, please contact the IT Service Desk on:
• Australia: 1800 033 534
• New Zealand: 0800 945 995

To reset your Microsoft AD password, visit https://portal.office.com on a PC
