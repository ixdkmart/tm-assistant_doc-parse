Glass Breakages

• Record Keycode/APN on breakage sheet.

• Use cut resistant gloves when placing into the bin.

• Always wrap glass fragments in a suitable material i.e. bubble wrap, before placing them in the dump master /skip bin.

• Do not push items down if bin is full (ask a manager to organize bin to be emptied).

• When emptying bin, ask for a 2-person lift, wear safety glasses.

• Action write off sheet daily as write offs on the PDT, Cross through items once actioned.
