"Waiting for DHCP" error

If a phone has reached any number above 200 on "waiting for DHCP", perform the following to reset the values of the phones.
Below should be the message you see on the screen

• Select * on the phone
• And enter the code 27238 and Select #
• Next scroll down until you see the option "Reset Values"
• After, leave the phone to boot until you are presented with the extension screen.
