Managing Offsite Storage
Learning Guide

To undertake this process, you will need:
1. To be in a controlled environment to ensure other product is not accidentally picked up
2. PDT
3. Hand wand
4. RFID portable printer
5. Offsite number template to label Pallets/FB2's/KC1's
6. Offsite control sheet

Managing offsite storage

1. Sign into the RFID app

2. You must be connected to the hand wand before you can complete this process. See instructions on the following pages on how to connect the hand wand.

Once you've logged in and connected to the hand wand:

3. Click the "OFFSITE" button from the main screen

4. On the offsite screen, two options will appear:
• Pack bundle
• Unpack bundle

Pack bundle is used to send stock to offsite
Unpack bundle is used to receive stock back from offsite

## Page 2

Relocating TO offsite

1. Press the "Pack Bundle" tile

2. Press "NEW" to create a new "bundle number"

3. Select a printer by choosing "Bundle Printer". E.g. 'Bundle Printer 21' will print the label to Printer 21

4. Press 'OK' to confirm your printer selection;
5 bundle labels should print:
• 1 label to be added to the stores control sheet
• 4 labels to be added to the pallet template for each side of the pallet/FB2/KC1

5. Press 'OK' once the bundle labels have successfully printed

6. Scan the bundle label's QR code with the PDT scanner or type in the bundle number

## Page 3

Relocating TO offsite continued

7. Ensure the RFID hand wand is connected to the PDT and the 'achievement' settings are set to 'auto'

8. Start hand scanning and see the 'Actual' count increase as new tags are recognised by the hand wand. Ensure you're in a controlled environment where other EPC's cannot be picked up accidentally.

9. Review the EPC tags read by the hand wand and remove any unwanted items by pressing the 'bin' icon

10. When happy with the EPC tags counted, press 'FINALISE'

11. Press 'OK' when the confirmation message displays, relocating your EPC tags to Offsite

## Page 6

Connecting the hand wand

1. With the hand wand:
• Connect the PDT to the hand wand
• Turn on the hand wand by holding in the 'power on' button (top right, green light)
• You will hear a beep when it turns on

2. Hold in the Bluetooth button (bottom left, flashing blue light) until the blue light is on – indicating the Bluetooth is available for connection

3. Open the RFID app and then log into it

4. Select the hand wand symbol in the bottom left hand corner

5. Select 'Reconnect' toggle to turn on/off hand wand that's connected or select 'Change device'

6. Ensure hand wand is turned on and Bluetooth is on
• Select 'START SEARCH'
