## Page 4

Waffle an' Pancake Guide - completin' a task  

5. Open a task t’ complete by selectin’ th’ task card  

6. Scan th’ station location barcode with yer device or type in th’ station number by pressin’ th’ keyboard icon.  

7. Once th’ station location be accepted, scan th’ barcode o’ th’ first ingredient in th’ location. Work from left t’ right, top t’ bottom.  

Scanned Ingredient Information:  

- Station location currently in progress  
- Th’ "In Progress" tab shows th’ most recently scanned ingredient info  
- Th’ "Take" number be th’ minimum quantity that should be used o’ this ingredient. If 0, scan th’ next item.  
- Th’ location/s that this ingredient be scanned into on th’ kitchen shelf an' in th’ storage room  
- Keyboard t’ type in an ingredient code if th’ barcode can’t be scanned  
- If ye be usin’ th’ last o’ this ingredient, make sure this be selected t’ also scan this ingredient out o’ this station location

## Page 5

Waffle an' Pancake Guide - completin' a task continued  

8. After scannin’ th’ barcode, th’ "Take" number indicates th’ minimum quantity ye should use. If there be less than that in th’ station, use all, matey.  

9. To continue, scan th’ barcode o’ th’ next ingredient in th’ station location.  
If th’ "Take" number be 0, this ingredient don’t need t’ be used an' ye can scan th’ next item.  

10. If th’ ingredient be used in more than one location, enter th’ quantity ye have taken from this location.  

11. Once each ingredient (one o’ each item) in th’ station location has been scanned an' all relevant quantities used, press "Finalise Preparation".  

12. Before finalisin’, make sure there be no ingredients in th’ station location that have been missed.  

13. Yer task be complete an' has been removed from th’ app home screen.  
Ye can now select th’ next task.

## Page 6

Troubleshootin’  

Refer t’ these steps when troubleshootin’ th’ followin’ error messages:  

Error - Invalid station barcode on task screen  
1. Th’ barcode scanned be not a station location barcode, try scannin’ th’ correct one that matches th’ task ye have selected.  

Error - Barcode scanned don’t match station selected on task screen  
1. Th’ scanned station location barcode don’t match th’ task ye have selected. Make sure ye be scannin’ th’ correct station.  

Error - Station already scanned on task screen  
1. Th’ correct station location barcode has already been scanned. Proceed with scannin’ an ingredient code.  

Error - Ingredient recall/withdrawal on task screen  
1. This ingredient has been recalled or withdrawn from use an’ should not be used in preparation.  
2. Make sure this ingredient be moved t’ th’ recall bin immediately.  
3. After removal, delete th’ ingredient from th’ station.

Troubleshootin’ continued  

Refer t’ these steps when troubleshootin’ th’ followin’ error messages:  

Errors - Somethin’ went wrong. Please try again  
or

1. Pull down th’ app t’ refresh  
2. Make sure ye be in network range an’ have Wi-Fi bars on th’ device  
3. If error message still be displayed, log out an' log back into th’ app  
4. If error message still be displayed after loggin’ back in, reboot device  
5. If rebootin’ th’ device don’t clear th’ error message, contact th’ IT Service Desk  

Error - lost connection with no tasks displayed on home screen  

1. Pull down th’ app t’ refresh

3. If error message still be displayed, log out an' log back in t’ th’ app
